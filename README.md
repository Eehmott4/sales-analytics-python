# 📊 Sales Analytics Pipeline in Python

Mini sistema de Business Intelligence desenvolvido em Python para análise e visualização de dados de vendas.

---

## 🚀 Objetivo

Construir um pipeline estruturado capaz de:

- Ler dados CSV
- Validar schema automaticamente
- Padronizar tipos e datas
- Calcular métricas de negócio
- Gerar análise temporal
- Produzir visualizações gráficas

---

## 🧠 Arquitetura

Pipeline:

Ingestão → Validação → Tratamento → Análise → Visualização
